Take YouTube Video Screenshots

Capture and save high-quality screenshots from YouTube videos with ease.

How to use

  • Download File : Left Click on Button
  • Download File with Prompt : Alt/Option + Left Click on Button
  • Copy to Clipboard : Shift + Left Click on Button
  • * Shutter sound/effect can be disabled in the add-on Preferences
  • * Download Method can be adjust in the add-on Prefereces
  • * Added "GIF Maker" button to create simple GIFs

Note

  • File will be automatically saved in your default download location with subfolder named "Youtube Screenshots" and split into folders based on video id
  • While creating the folder, if there is an illegal characters in the video title, The folder name contains only the video id
  • * The image quality is directly related to the current video quality you have selected
